# Bobolink Construction LLC Website V2

A static, mobile-responsive, vintage catalog-inspired landing page.

## Files

Upload these files to the root of your GitHub repository:

- index.html
- style.css
- script.js
- images/

## Required image filenames

Put these exact files inside the `images` folder:

- hero-catio.jpg
- catio-1.jpg
- catio-2.jpg
- catio-3.jpg
- fence.jpg
- gate.jpg
- repair.jpg
- deck.jpg
- carpentry.jpg
- icon-leaf.jpg
- icon-weather.jpg
- icon-tree.jpg
- icon-shield.jpg
- icon-tools.jpg
- icon-bobolink.jpg

GitHub Pages is case-sensitive, so filenames must match exactly.
